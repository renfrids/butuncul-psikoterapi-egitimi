# Bütüncül Psikoterapi Eğitimi — Açılış Sayfası

Bütüncül Terapiler Enstitüsü / Banu İkincisoy **Bütüncül Psikoterapi Eğitimi** tanıtım sayfası.
Tek sayfalık, statik (build adımı yok). Dönüşüm hedefi: sayfa içindeki bilgi/ön başvuru formu.
Ödeme sayfadan alınmaz.

## Çalıştırma

Sunucu gerekmez; `index.html`'i çift tıklayıp tarayıcıda açabilirsin.

VS Code'da geliştirirken **Live Server** eklentisi önerilir (otomatik yenileme):

1. VS Code'da bu klasörü aç.
2. `Live Server` eklentisini kur.
3. `index.html` → sağ tık → **Open with Live Server**.

## Yapı

```
index.html        Sayfa iskeleti (bölümler HTML yorumlarıyla ayrılmış)
css/styles.css    Stiller (:root içinde tasarım tokenları)
js/main.js        Etkileşimler (menü, akordeon, sticky bar, form)
assets/           Logolar + görseller (assets/images/)
CLAUDE.md         Claude Code için proje bağlamı ve TODO listesi
```

## Sırada ne var?

Ayrıntılar `CLAUDE.md` içinde. Özetle: form gönderimini bir hedefe bağlamak (e-posta /
Google Sheets / backend), tanıtım videosunu gömmek, gerçek görselleri ve eğitmen biyografisini
eklemek, kesinleşen tarih/ücret/kontenjan bilgilerini doldurmak.
